INTRODUCTION

A Computer Network is a data-communication network that consists of computers (called nodes), and connections between them (called links).
A network can be mathematically represented as a graph where the nodes represent the vertices of the graph and the links represent the edges. Consequently, all the algorithms that are applicable to graphs (such as Bellman Ford, Dijkstra, Prim’s, Kruskal) will be applicable to networks as well and we’ll use these quite frequently to solve various problems that arise in the networks.



Five main components of any general data communication system are:

Message: The information to be communicated. It can be text, numbers, pictures, audio, video etc.
Sender: A device that sends the data message.
Receiver: A device that receives the message.
Transmission Medium: Physical path by which the data travels. Some examples are twisted wire cables, coaxial cables, optical fibres, or radio waves in Wi-Fi.
Protocol: A set of rules that govern the data communication. A protocol defines the format and order of messages exchanged between two or more communicating entities, as well as the actions taken on the transmission and/or receipt of a message or other event.


Transmission modes between two devices can be categorized into 3 types:


Simplex:Communication is unidirectional. Keyboards and Traditional monitors are examples of simplex communication. This mode uses the entire capacity of the channel to send data in one direction.

Half-duplex: Each station can both transmit and receive, but not at the same time. Walkie Talkies and Citizen Band Radios are examples of half-duplex communication. Entire capacity of channel is utilized for each direction.

Full-duplex: Both stations can transmit and receive at the same time. An example for the full duplex communication is the telephone network. The capacity of the channel must be divided between the two directions.
